# C++ Playground

## fast development environment
requirements
* nodejs
* npm

compiles and runs the code when the main.cpp or in (input file) changes
the output is written to out (output file) and the errors in terminal
```bash
$ npm i -g nodemon
$ nodemon
```
> ⚠️ don't write to an output file because if you cause a infinite loop your system will crash. So for that remove ***"`<out`"*** from ***nodemon.json***
